В личном кабинете работодателя отображается:
- информация о компании: наименование, краткое описание, сфера деятельности, ссылки на сайт и соцсети. Дополнительное наполнение (например, фото офиса, видеопрезентация);
- форма создания новой возможности (вакансии, стажировки, мероприятия), которая дублирует поля наполнения карточки возможности;
- список активных, закрытых и запланированных возможностей с возможностью редактирования;
- список откликнувшихся соискателей с возможностью просмотра их профилей и изменения статуса отклика (принят/отклонен/в резерве);
- функционал поиска и фильтрации по своим вакансиям и откликам.
