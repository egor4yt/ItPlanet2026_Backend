На странице регистрации доступен выбор роли на платформе — работодатель или соискатель. Предварительная регистрация по любой роли требует адрес электронной почты, отображаемое имя и пароль. В зависимости от роли куратором может быть запрошена дополнительная информация. В случае выбора роли «работодатель» при регистрации необходима верификация компании.
