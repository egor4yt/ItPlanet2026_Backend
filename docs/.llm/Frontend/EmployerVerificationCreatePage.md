Реализуй вёрстку страницы EmployerVerificationCreatePage, твоя задача сделать только вёрстку и стилизацию, без бизнес-логики, если нужны будут какие-то типы, то создавай их прям в файле вёрстки.
Необходимо реализовать:
- поле для ввода длинного текста: сообщение от компании
- поле для ввода ссылки: соцсеть
- тип документа - select с возможностью выбрать один элемент, на основе которого компания запрашивает верификацию 
- кнопку "загрузить документ", по нажатию на которую пользователь выберет один файл со своего устройства
- кнопку "отправить заявку"

В качестве примера используй такой тип:
```json
{
  "verificationTypeId": 0,
  "requestMessage": "string",
  "socialNetworkLink": "string",
  "taxpayerIndividualNumber": "string"
}
```
requestMessage - сообщение от компании
taxpayerIndividualNumber - номер ИНН
socialNetworkLink - ссылка на соцсеть компании
verificationTypeId - id типа подтверждающего файла для верификации компании
