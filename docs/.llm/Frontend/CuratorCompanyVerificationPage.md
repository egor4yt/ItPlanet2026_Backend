Реализуй вёрстку страницы CuratorCompanyVerificationPage, твоя задача сделать только вёрстку и стилизацию, без бизнес-логики, если нужны будут какие-то типы, то создавай их прям в файле вёрстки.
Необходимо отобразить имя и ИНН компании, сообщение от компании, предыдущий ответ куратора, время последнего ответа в формате "20 мая 2025 21:40", кнопку "соцсеть" и кнопку "профиль компании", тип документа, на основе которого компания запрашивает верификацию и  кнопку "скачать документ" (При нажатии на кнопку "Скачать документ" необходимо скачивать файл, в названии которого будет написано "{Имя компании} {Способ верификации}.pdf", файл должен быть пустым).
Также необходимо функционал ответа куратора, возможно три сценария:
1. Отказ в верификации, в таком случае куратор должен описать текстом причину отказа
2. Запрос дополнительных данных, в таком случае куратор должен описать текстом чего не хватает
3. Подтверждение, в таком случае ничего дополнять не нужно

В качестве примера входных данных используй такой тип:
```json
{
  "requestMessage": "string",
  "responseMessage": "string",
  "taxpayerIndividualNumber": "string",
  "socialNetworkLink": "string",
  "changedOn": "2026-03-24T14:35:04.931Z",
  "status": {
    "id": 0,
    "title": "string"
  },
  "type": {
    "id": 0,
    "title": "string"
  }
}
```
requestMessage - сообщение от компании
responseMessage - ответ куратора на предыдущую заявку
taxpayerIndividualNumber - номер ИНН
socialNetworkLink - ссылка на соцсеть компании
changedOn - последние дата и время изменения заявки в UTC
status.title - текущий статус заявки
type.title - тип подтверждающего файла для верификации компании
